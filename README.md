Скрипт, который считывает логи из Steam и выводит информацию о загрузке
1. Считывает скорость загрузки
2. Выводит в консоль скорости загрузки игры каждую 1 минуту (на протяжении 5 минут)
3. Учитывает статус загрузки
4. А также показывает, какая игра скачивается
   
Можно посмотреть лог файл **steam_monitor.log**, здесь наглядно показываются логи
